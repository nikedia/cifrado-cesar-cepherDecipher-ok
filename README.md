**#PROYECTO CIFRADO CESAR**

Esta aplicación te ayuda a cifrar una frase basado en el código ASCII y devolviendo esa frase cifrada al alfabeto normal.

**PSEUDOCÓDIGO**

funcion cipherDecipher

 Se crea la variable str
